# Leistungsbeschreibung

## Prompts für die Erstellung von Leistungsbeschreibungen

### Prompt: Funktionale Leistungsbeschreibung
```
Erstelle eine funktionale Leistungsbeschreibung für folgendes Vorhaben:

[BESCHAFFUNGSGEGENSTAND]

**Struktur:**
1. **Ausgangslage und Zielsetzung**
   - Ist-Situation
   - Soll-Zustand
   - Projektziele

2. **Funktionale Anforderungen**
   - Kernfunktionen
   - Erweiterte Funktionen
   - Performance-Anforderungen

3. **Nicht-funktionale Anforderungen**
   - Qualitätsanforderungen
   - Sicherheitsanforderungen
   - Usability-Anforderungen

4. **Schnittstellen und Integration**
   - Systemlandschaft
   - Datenformate
   - Protokolle

5. **Rechtliche und normative Vorgaben**
   - Gesetzliche Anforderungen
   - Standards und Normen
   - Compliance-Vorgaben

**Formulierung:** Ergebnisorientiert, technologieneutral, prüfbar
```

### Prompt: Technische Spezifikation
```
Erstelle eine detaillierte technische Spezifikation:

**EINGABE:** [Funktionale Anforderungen]

**AUFBAU:**
1. **Systemarchitektur**
   - Gesamtarchitektur
   - Komponenten
   - Schnittstellen

2. **Hardware-Anforderungen**
   - Mindestanforderungen
   - Empfohlene Ausstattung
   - Skalierbarkeit

3. **Software-Anforderungen**
   - Betriebssystem
   - Laufzeitumgebung
   - Abhängigkeiten

4. **Datenmodell**
   - Datenstrukturen
   - Datenbankanforderungen
   - Backup/Recovery

5. **Sicherheitsanforderungen**
   - Authentifizierung
   - Autorisierung
   - Verschlüsselung
   - Auditierbarkeit

**Prüfkriterien:** Messbare, nachvollziehbare Anforderungen
```

### Prompt: Qualitätskriterien definieren
```
Definiere umfassende Qualitätskriterien für:

[LEISTUNGSGEGENSTAND]

**Qualitätsdimensionen:**
1. **Funktionale Qualität**
   - Vollständigkeit
   - Korrektheit
   - Zuverlässigkeit

2. **Technische Qualität**
   - Performance
   - Skalierbarkeit
   - Wartbarkeit
   - Portabilität

3. **Benutzerqualität**
   - Usability
   - Accessibility
   - User Experience

4. **Betriebsqualität**
   - Verfügbarkeit
   - Monitoring
   - Support

5. **Dokumentationsqualität**
   - Vollständigkeit
   - Verständlichkeit
   - Aktualität

**Format:** Pro Kriterium: Definition, Messverfahren, Zielwert, Mindestanforderung
```
