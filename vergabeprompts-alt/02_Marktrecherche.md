# Marktrecherche

## Prompts für die Marktrecherche

### Prompt: Suchbegriffe generieren
```
Du bist Experte für öffentliche Beschaffung. Generiere relevante Suchbegriffe für eine umfassende Marktrecherche.

Beschaffungsgegenstand: [EINGABE]

Erstelle:
1. **Hauptsuchbegriffe** (deutsch und englisch)
2. **Synonyme und Varianten**
3. **Branchenspezifische Begriffe**
4. **CPV-Code Zuordnung**
5. **Erweiterte Suchbegriffe** (verwandte Produkte/Dienstleistungen)

Format: Strukturierte Liste mit Begründung der Relevanz
```

### Prompt: Anbieteranalyse
```
Analysiere systematisch potentielle Anbieter für folgenden Beschaffungsgegenstand:

[BESCHAFFUNGSGEGENSTAND]

**Analyseschema:**
1. **Anbieteridentifikation**
   - Firmenname und Rechtsform
   - Standort und Regionalität
   - Unternehmensgroesse

2. **Leistungsportfolio**
   - Kernkompetenzen
   - Produktspektrum
   - Spezialisierungen

3. **Marktposition**
   - Marktanteil
   - Referenzen (öffentliche Auftraggeber)
   - Zertifizierungen

4. **Vergaberelevante Bewertung**
   - Lieferfähigkeit
   - Zuverlässigkeit
   - Innovationskraft

Erstelle eine Übersichtstabelle mit Bewertung (1-5 Sterne).
```

### Prompt: Technologie-Screening
```
Führe ein Technologie-Screening durch für:

[TECHNOLOGIEBEREICH]

**Screening-Dimensionen:**
1. **Aktueller Technologiestand**
   - Etablierte Lösungen
   - Marktreife
   - Standardisierung

2. **Emerging Technologies**
   - Neue Entwicklungen
   - Zukunftspotential
   - Reifegrad

3. **Kompatibilität und Integration**
   - Schnittstellen
   - Legacy-Systeme
   - Interoperabilität

4. **Wirtschaftlichkeit**
   - Investitionskosten
   - Betriebskosten
   - ROI-Potentiale

5. **Risikobewertung**
   - Technische Risiken
   - Vendor Lock-in
   - Obsoleszenz

Empfehlung: Technologiestrategie für Ausschreibung
```
