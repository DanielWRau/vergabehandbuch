# Vergabeunterlagen

## Prompts für die Erstellung von Vergabeunterlagen

### Prompt: Bekanntmachung erstellen
```
Erstelle eine EU-Bekanntmachung für folgendes Vergabeverfahren:

**EINGABEN:**
- Auftraggeber: [NAME]
- Beschaffungsgegenstand: [LEISTUNG]
- Geschätzter Auftragswert: [WERT]
- Verfahrensart: [VERFAHREN]

**STRUKTUR:**
1. **Auftraggeber-Informationen**
   - Vollständige Anschrift
   - Kontaktdaten
   - Vergabestelle

2. **Auftragsbeschreibung**
   - CPV-Code
   - Kurzbeschreibung
   - Leistungsort
   - Vertragslaufzeit

3. **Verfahrensinformationen**
   - Verfahrensart
   - Zuschlagskriterien
   - Fristen

4. **Teilnahmebedingungen**
   - Eignungskriterien
   - Nachweise
   - Ausschlussgründe

**Formulierung:** Präzise, vollständig, rechtssicher
```

### Prompt: Bewertungsmatrix entwickeln
```
Entwickle eine umfassende Bewertungsmatrix für:

[BESCHAFFUNGSVORHABEN]

**Bewertungsstruktur:**
1. **Preisbewertung (X%)**
   - Bewertungsmethode (z.B. Richtwertmethode)
   - Berechnungsformel
   - Maximalpunktzahl

2. **Qualitätsbewertung (Y%)**
   - Hauptkriterien
   - Unterkriterien
   - Gewichtung
   - Bewertungsskala (z.B. 0-5 Punkte)

3. **Bewertungskriterien definieren:**
   - Technische Qualität
   - Methodische Qualität
   - Soziale Nachhaltigkeit
   - Umweltkriterien
   - Innovation

**Pro Kriterium angeben:**
- Gewichtung
- Bewertungsmaßstab
- Beispiele für Punktvergabe
- Mindestanforderungen

**Gesamtgewichtung:** Preis/Qualität = X% / Y%
```

### Prompt: Vertragsbedingungen formulieren
```
Formuliere Vertragsbedingungen für:

[VERTRAGSGEGENSTAND]

**VERTRAGSBAUSTEINE:**
1. **Leistungsbestimmung**
   - Leistungsumfang
   - Leistungszeit
   - Leistungsort
   - Änderungsvorbehalte

2. **Vergütung und Zahlung**
   - Vergütungsart
   - Zahlungsmodalitäten
   - Nachtragsregelungen
   - Preisanpassungen

3. **Durchführungsbestimmungen**
   - Projektorganisation
   - Berichtswesen
   - Qualitätssicherung
   - Abnahme

4. **Rechtliche Bestimmungen**
   - Gewährleistung
   - Haftung
   - Datenschutz
   - Geheimhaltung

5. **Besondere Bestimmungen**
   - Sicherheitsanforderungen
   - Umweltauflagen
   - Soziale Kriterien

**Formulierung:** Eindeutig, ausgewogen, durchsetzbar
```
